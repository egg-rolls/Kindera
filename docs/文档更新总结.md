## 技术栈变化

### 旧版本（v1.0）
- Java 8 控制台应用程序
- MySQL 8.x
- JDBC直连
- 分层架构（View → Service → DAO → Entity）

### 新版本（v2.0）
- **后端：** Spring Boot 2.7.18 / Java 8 / MySQL 8.0
- **前端：** Vue 3 / Vite / Element Plus
- **部署：** Docker Compose（MySQL + Spring Boot + Nginx）
- **架构：** 前后端分离（REST API）

## 主要新增功能

1. **REST API接口** - 8个Controller提供完整的REST API
2. **Vue 3前端** - 7个页面组件，响应式设计
3. **Docker部署** - 一键部署，包含健康检查
4. **Nginx反向代理** - 静态资源托管，API代理
5. **跨域配置** - CorsConfig支持前后端分离
